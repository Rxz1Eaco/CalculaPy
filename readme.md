# CalculaPY

- Este repositório contém uma *calculadora feita em python*.

- Idea surgiu no estágio

## Objetivo

- Facilidade de Uso: Fornece uma interface fácil de usar para que os usuários possam inserir dados e receber resultados de maneira intuitiva.

- Automatização de Cálculos: Realizar cálculos matemáticos de forma rápida e precisa, automatizando tarefas que, de outra forma, seriam demoradas ou suscetíveis a erros humanos.

- Flexibilidade: Permitir a realização de uma variedade de operações matemáticas, como adição, subtração, multiplicação, divisão e exponenciação até agora

## Conteúdo do Repositório

A organização do repositório está da seguinte forma:

- **`.py`**: Arquivos python que detalham o processo feito em uma calculadora
